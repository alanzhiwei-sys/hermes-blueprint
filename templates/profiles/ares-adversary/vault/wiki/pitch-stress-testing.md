# Pitch Stress-Testing Framework

> **Use when:** Reviewing proposals, quotations, sales decks, or marketing claims before they reach a real client. This is your pre-flight checklist. If a claim fails any test, fix it or cut it.

---

## Phase 1: Proposal Weakness Audit (5 Questions)

Before you even look at pricing, audit the structural integrity:

1. **What is this proposal NOT saying?** Every proposal has omissions. Find them. Are you silent on timeline? Permits? Defect liability? The client will fill silence with worst-case assumptions.

2. **What does the client lose by saying no?** If the answer is "nothing urgent," your BATNA asymmetry is dangerous. A client with no deadline pressure has all the power.

3. **Which line item would you cut if you were the client?** Identify the fattest, least-defended item. That's where the client will attack. Pre-defend it or pre-cut it.

4. **Where does the proposal assume trust instead of earning it?** "We use quality materials" — says everyone. "We use Blum hinges (Austria), Hafele track (Germany), and provide batch photos of materials on arrival" — different story.

5. **What's the hidden anchor?** A price of $48,000 next to a "competitor estimate" of $65,000 is an anchor play. Is the $65k real, verifiable, and current — or fabricated social proof?

---

## Phase 2: Claim Verification Checklist

For every claim in the proposal, answer:

| Claim Example | Verification Test | Pass? |
|---|---|---|
| "Premium European materials" | Can I name the brand, country of origin, and show a photo of the packaging? | |
| "Completed 200+ HDB projects" | Can I produce a list (anonymised) with project types and dates? | |
| "5-year workmanship warranty" | Is it in the contract? What exactly does it cover? What are the exclusions? | |
| "Project completed in 6 weeks" | Is that calendar weeks? Working days? Does it include permit wait time? | |
| "Best price guaranteed" | Against whom? What's the matching process? What's excluded? | |
| "[YOUR_BUSINESS_ALT] certified installer" | Certified by whom? Can I show the certificate? Is it current? | |

**Rule:** If you can't verify a claim in under 60 seconds, cut it. Unverifiable claims become weapons in the client's hands — they'll ask for proof, and when you can't produce it instantly, you lose credibility on everything else.

---

## Phase 3: Pricing Transparency Test

Clients don't distrust high prices — they distrust prices that feel arbitrary.

**The "Explain This Number" Test:** Point to any line item at random. Can you explain exactly what drives that cost in 2 sentences?

- **Fail:** "Carpentry — $18,500" (lump sum, no breakdown)
- **Pass:** "Carpentry — $18,500: kitchen cabinets ($8,200 — 24ft run, plywood carcass, Blum soft-close), master bedroom wardrobe ($5,800 — 8ft, sliding doors, full-height), plus 2 bathroom vanities ($4,500)"

**The "Why Not Cheaper" Test:** For major items, add one line: "We could do this for $X less if we used [cheaper material/method], but we recommend against it because [consequence]." This pre-empts the client's "can't you do it cheaper" by showing you've already considered and rejected the cheap option for good reasons.

## Phase 4: Competitor Comparison Gap Analysis

Map your proposal against the likely competitor landscape:

| Dimension | Us | Competitor A (Value) | Competitor B (Premium) | Gap / Risk |
|---|---|---|---|---|
| Price | | | | |
| Materials spec | | | | |
| Timeline | | | | |
| Warranty | | | | |
| Smart home integration | | | | |
| After-service support | | | | |

**Critical questions:**
- Where are we in the middle? (Danger zone — neither cheap enough nor premium enough)
- What's our one dimension of clear superiority? (If you can't name it, you don't have a pitch)
- Where could a competitor beat us with one sentence? ("They don't even include ______")

---

## Phase 5: The "So What?" Test

Go through every feature claim and ask: **So what?**

```
FEATURE: "We use German-engineered Hafele drawer tracks."
SO WHAT? "So your kitchen drawers will open and close smoothly for 15+ years, not start jamming in year 3 like the generic tracks most contractors use."

FEATURE: "All our electricians are EMA-licensed."
SO WHAT? "So your electrical work passes HDB/BCA inspection first time, no re-wiring, no delay to your move-in date."

FEATURE: "[YOUR_BUSINESS_ALT] lighting scenes are pre-programmed."
SO WHAT? "So on move-in day your 'Dinner,' 'Movie,' and 'Goodnight' scenes work with one tap — no 2-week fiddling with apps and remotes while living in a construction zone."
```

**Rule:** If a feature can't answer "so what?" with a concrete, client-relevant benefit, delete it. Features without benefits are filler — and filler makes clients suspicious.

---

## Scoring Your Pitch

| Score | Meaning | Action |
|---|---|---|
| 5/5 phases clean | Battle-ready | Deploy |
| 3-4 phases clean | Has vulnerabilities | Fix before client presentation |
| 1-2 phases clean | High risk of losing deal to objections | Redraft before showing anyone |
| 0/5 | The client will destroy this | Start over |
