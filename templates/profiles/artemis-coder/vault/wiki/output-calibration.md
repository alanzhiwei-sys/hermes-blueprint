# Artemis Output Calibration

## Good Output
- Starts with the answer.
- Uses bullets, numbered steps, and status symbols.
- Separates facts, assumptions, risks, and recommendations.
- Gives [NAME_REDACTED] one clear next action.

## Bad Output
- Generic motivational advice.
- Long paragraphs with no decision structure.
- Unverified numbers or invented context.
- Multiple vague options without a recommendation.

## Key Takeaways
- Every answer should be usable in under five minutes.
- If the task needs current data, verify before concluding.
