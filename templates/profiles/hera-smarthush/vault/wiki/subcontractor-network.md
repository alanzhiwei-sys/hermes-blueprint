## Required Trades & Specialisations

### 1. Licensed Electrician (EMA-Licensed)
- **Role**: Electrical works related to smart home installation — switch replacement, neutral wire availability check, DB box assessment, dedicated circuit installation where needed
- **Licence Required**: EMA (Energy Market Authority) Electrical Worker Licence — minimum Grade 7 (Electrician)
- **Scope**:
  - Assess and confirm neutral wire availability at switch boxes
  - Replace existing switches with smart switches (Aqara, Zooz)
  - Install new power points for hubs, network gear, motorised blinds
  - Install dedicated circuits for high-draw automation equipment
  - Earth leakage testing post-installation
  - LEW (Licensed Electrical Worker) sign-off for any new circuit additions
- **Cannot Do**: Low-voltage data cabling (separate cert); programming/configuration ([YOUR_BUSINESS_ALT] scope)

### 2. Structured Cabling Technician (Low-Voltage)
- **Role**: Cat6/Cat6a cable runs, termination, testing, and certification
- **Certification**: Fluke DSX cable certifier or equivalent; ideally BICSI-certified
- **Scope**:
  - Run Cat6/Cat6a cables from network rack to access point locations, camera positions, and key data points
  - Terminate cables to RJ45 keystone jacks and patch panels
  - Certify every cable run (Fluke test report — not just a basic continuity tester)
  - Install trunking/conduit for exposed cable runs (coordinate with renovation contractor for concealed runs)
  - Label all cables at both ends with unique IDs
- **Key Requirement**: Every Cat6 run must pass Fluke Permanent Link certification. No exceptions. Wi-Fi-dependent devices (cameras, access points) are only as good as the cable feeding them.

### 3. AV / Mounting Installer
- **Role**: Physical mounting of devices — cameras, access points, sensors, tablet wall mounts, motorised blind brackets
- **Scope**:
  - Ceiling-mount PoE cameras and access points (coordinate with false ceiling contractor)
  - Wall-mount tablet (for Home Assistant dashboard)
  - Sensor mounting — door/window sensors, motion sensors (adhesive + screw where possible)
  - Motorised blind installation — bracket mounting, power connection, alignment
  - Scene keypad installation and physical alignment
  - Cable management — trunking, cable ties, labelling
- **Key Requirement**: Must understand that camera and AP placement is critical to system performance. Must follow [YOUR_BUSINESS_ALT] placement plans exactly, not "wherever is convenient."

## Vetting Process

### Stage 1: Credentials Check
- **Electrician**: Verify EMA licence number at EMA e-Licence Info portal; check expiry date
- **Cabler**: Verify BICSI certification or equivalent; request sample Fluke test report
- **AV Installer**: Portfolio review — photos of 5+ camera/AP installations; check for neat cable management

### Stage 2: Test Job
- Assign a small scope (e.g., single AP + 2 camera cable runs, or 5 switch replacements)
- Evaluate on criteria below

### Stage 3: Evaluation Matrix
| Criteria | Weight | Description |
|----------|--------|-------------|
| Quality of Work | 35% | Cable certification passes, switch alignment, trunking neatness |
| Reliability | 25% | Shows up on time, completes within quoted duration |
| Communication | 20% | Flags issues early, responds to WhatsApp within 2 hours during working hours |
| Cleanliness | 10% | Clears debris, vacuums after drilling, no cable offcuts left behind |
| Documentation | 10% | Provides cable test reports, photo documentation of completed work |

**Pass Threshold**: ≥ 7.5/10 overall; no single criterion below 5/10
**Probation**: First 3 jobs on probationary status; weekly check-ins with Hera

## Engagement Protocol

### Pre-Job
1. Send scope document with floor plan, device schedule, and cable schedule at least 5 working days before start
2. Confirm availability, rates, and estimated duration in writing
3. Verify licences/certifications are current (don't assume — check every job)
4. Confirm insurance coverage is active (public liability minimum $500K)

### During Job
1. Morning check-in: confirm on-site arrival, flag any access issues
2. Mid-job inspection ([YOUR_BUSINESS_ALT] lead performs spot check of work quality)
3. All cable runs documented with photos before concealment (wall/ceiling closure)
4. Fluke test reports provided same day as termination

### Post-Job
1. Defect/warranty period: 30 days on workmanship
2. Cable re-test included if connectivity issues reported within warranty period
3. Quality score updated in subcontractor database
4. Invoice reviewed against quoted scope and approved within 14 days

## Subcontractor Red Flags
- [ ] Cannot provide valid licence/certification on request
- [ ] Reluctant to provide Fluke test reports ("trust me, the cable works")
- [ ] Frequently reschedules or arrives late without notice
- [ ] Pushes for shortcuts ("no need trunking, just run the cable on the false ceiling")
- [ ] Leaves debris/offcuts on site
- [ ] Substitutes materials without approval (different cable grade, different trunking)
- [ ] Quotes without site visit for complex jobs (camera/AP placement especially)

## Backup Subcontractor Pool
Always maintain at least 2 qualified electricians and 2 cabling techs in the approved pool. For critical-path installations, have a backup contact ready. Subcontractor unavailability is not an acceptable reason for project delay.
