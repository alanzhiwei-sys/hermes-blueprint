# Hermes Blueprint — First Business Win Catalog

The first 48 hours decide whether Hermes Blueprint feels valuable or like another tool to configure.

Every beta user must choose one first business win and create one usable artifact.

## Activation Promise

> Within 48 hours, Hermes should help you create one business artifact you can actually use.

Not a demo. Not a toy prompt. A usable business asset.

## Recommended First Wins

| First Win | Best For | Output Artifact | Value Signal |
|---|---|---|---|
| **Sales follow-up system** | Leads come in but follow-up is inconsistent | 7-day WhatsApp/email follow-up sequence | Faster replies, fewer forgotten leads |
| **Customer FAQ / response library** | Owner repeats the same answers daily | Reusable answer bank with warm/professional replies | Less repeated typing, more consistent tone |
| **Weekly owner command briefing** | Owner feels scattered across tasks, money, risks, follow-ups | Weekly review template and prompt | Better control over business priorities |
| **SOP / checklist builder** | Delivery quality depends too much on memory | Step-by-step checklist with approval points | Fewer mistakes, easier delegation |
| **Competitor / pricing research desk** | Pricing or positioning is unclear | Research brief with competitors, gaps, recommended moves | Better pricing confidence |
| **Proposal / quote assistant** | Quotes/proposals take too long or lack clarity | Quote improvement checklist + client-facing explanation drafts | Faster, sharper proposals |

## Segment-Specific Examples

### Renovation / ID / Contractor

First win: **Lead qualification + quote follow-up system**

Output:

- Enquiry intake questions
- WhatsApp first reply
- Site visit qualification checklist
- Day 1 / Day 3 / Day 7 follow-up messages
- Red-flag client filter

Prompt:

```text
Act as my renovation sales coordinator. Build a lead qualification and follow-up system for HDB/condo renovation enquiries in Singapore. Include WhatsApp replies, site visit questions, budget/scope filters, and a 7-day follow-up sequence. Keep messages warm, practical, and not pushy.
```

## Beta Validation Questions

Ask these after the first 7 days:

1. Did you create a usable artifact within 48 hours?
2. Did you reuse it in real work?
3. Did it save time, improve quality, or reduce decision stress?
4. Would you pay for this workflow alone?
5. What was confusing before the artifact was created?
6. Which part felt like "AI hype" instead of practical help?

## Pass / Fail Standard

### Pass

The user says one of these:

- "I used this in my actual business."
- "This saved me time."
- "This made my process clearer."
- "I know what to use Hermes for next."

### Fail

The user says:

- "Cool, but I don't know what to do with it."
- "It feels like prompts/files."
- "I didn't create anything I can use."
- "I still prefer just opening ChatGPT."

If a beta user fails this test, fix onboarding before recruiting more users.
